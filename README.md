# Синтаксический анализатор

Веб-приложение для синтаксического анализа выражений методом операторного предшествования.

## Описание

Проект реализует синтаксический анализатор для грамматики варианта 13, который выполняет:
- **Лексический анализ** — разбиение входной строки на лексемы (идентификаторы и шестнадцатеричные числа)
- **Синтаксический анализ** — построение дерева вывода методом операторного предшествования
- **Визуализацию** — отображение дерева разбора и процесса анализа

## Грамматика

```
S → a := F;
F → F + T | T
T → T * E | T / E | E
E → (F) | -(F) | a
```

## Лексемы

- Идентификаторы (буквы, цифры, подчеркивания)
- Шестнадцатеричные числа (формат `0xFF`, `0x1A` и т.д.)
- Операторы: `:=`, `+`, `*`, `/`, `-`, `(`, `)`, `;`

## Использование

Откройте `index.html` в браузере и введите выражение для анализа, например:
- `x := 0xFF;`
- `result := var1 + 0x2A * 0xB;`
- `val := -(0x1 * (y + 0x2));`

## Функции

- Ввод выражения для анализа
- Отображение найденных лексем
- Построение дерева вывода
- Матрица предшествования
- Лог процесса разбора (стек, входная цепочка, действия)
